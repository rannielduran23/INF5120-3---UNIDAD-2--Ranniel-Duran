# INF5120-3---UNIDAD-2--Ranniel-Duran
Aplicación de escritorio desarrollada en C# con Windows Forms que implementa una calculadora básica. El programa permite realizar operaciones matemáticas fundamentales (suma, resta, multiplicación y división) mediante una interfaz gráfica simple e intuitiva.

Windows Forms es una herramienta con la que puedes crear aplicaciones de escritorio con ventanas, botones y todo lo visual, usando el lenguaje C# de una forma bastante directa. Con formularios, etiquetas, cajas de texto y botones, armas programas con los que la gente puede interactuar y que reaccionan a lo que hace el usuario. Además, es ideal para ir agarrandole la mano a la programación orientada a objetos, porque mezclás la lógica, las validaciones y el diseño visual todo en un mismo proyecto.

De práctica, hice una calculadora básica con Windows Forms. La app suma, resta, multiplica y divide. El usuario escribe dos números en unos cuadros de texto, elige qué operación quiere con los botones, y el programa se fija que lo que ingresó sean números de verdad, muestra el resultado y listo. Hacer esta calculadora me ayudó a entender cómo funcionan los eventos, a diseñar una interfaz más o menos decente y a ver cómo se arma una aplicación de escritorio de principio a fin.
